Lotto Smash Android 앱

Lotto Smash는 로또 추첨 정보를 쉽고 빠르게 확인하고, 번호 생성부터 통계 분석까지 간편하게 사용할 수 있는 Android 앱입니다.

앱 소개

이 앱은 한국 로또 사용자를 위해 설계되었습니다. 직관적인 UI와 빠른 응답성을 갖추었으며, 로또 번호 생성과 결과 확인, 당첨 번호 분석을 한 곳에서 제공합니다.

핵심 기능

  • 로또 번호 자동 생성 및 커스텀 번호 저장
  • 최근 당첨 번호 통계 및 패턴 분석
  • 당첨 결과 알림 및 즐겨찾기 기능
  • 간단한 화면 구성으로 누구나 쉽게 사용 가능

배포 버전별 요약

버전 요약
1.3 문의 답변 알림 배지 및 알림 목록 표시. 문의·추천 이력 다중삭제 기능. 5개 화면 하단 배너 광고 추가. FCM 포그라운드 알림 방식 개선(스낵바→뱃지).
